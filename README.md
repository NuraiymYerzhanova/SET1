## Description

This Android application was created for Lab 1.  
It includes three basic Java Android programs: Hello World, Introduce Yourself, and a Number Guessing Game.

## Features

- Displays "Hello World" on the screen.
- Allows the user to enter a name and displays "Witaj <name>".
- Shows "Przedstaw się." when the name input is empty.
- Implements a number guessing game with random numbers from 1 to 10.
- Validates empty, non-integer, and out-of-range input.
